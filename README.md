# 🌌 Os Guardiões da Galáxia

Este projeto foi desenvolvido como parte dos meus estudos e tem como tema **Os Guardiões da Galáxia**, filme da Marvel que eu gosto muito.  
A ideia foi unir aprendizado em desenvolvimento web com minha paixão pelo universo dos Guardiões, criando um site inspirado nos personagens e histórias.

---

## 🚀 Objetivo
- 📚 Praticar conceitos de **HTML, CSS e JavaScript**  
- 🎨 Criar um site temático inspirado em um filme que admiro  
- 💡 Exercitar criatividade e design web  

---

## 🛠️ Tecnologias utilizadas
- 🖥️ **HTML5**  
- 🎨 **CSS3**  
- ⚡ **JavaScript**  

---

## 🎬 Personagens representados
- 🌟 **Star-Lord** (Peter Quill)  
- 🌳 **Groot**  
- 🦝 **Rocket Raccoon**  
- 🗡️ **Gamora**  
- 💪 **Drax, o Destruidor**  

---

## 📂 Estrutura do projeto
- 📄 `index.html` → Página principal  
- 🎨 `style.css` → Estilos e layout  
- ⚡ `script.js` → Interatividade  

---

## ✨ Conclusão
Este trabalho representa a união entre **estudo e paixão**: aprender desenvolvimento web enquanto crio algo inspirado em um filme que me marcou.  
🌌💫 *Eu sou Groot!* 🌳
